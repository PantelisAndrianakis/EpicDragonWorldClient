# Epic Dragon World Client
Unity3D client that connects with .NET Core server.

## Progress
- Connect to .NET Core server.
- Packet communication.
- Network encryption.
- Music manager.
- Server authentication.
- Support for UMA.
- Character management.
- Broadcast game object movement.
- Chat and messaging.

### Unity version.
2018.3.12f1
