# Epic Dragon World Client
Unity3D/C# client that connects with Java/Netty server.

## Progress
- Connect to Java server.
- Packet communication.
- Network encryption.
- Scene fader.
- Music manager.
- Server authentication.
- Character management.
- Broadcast game object movement.
